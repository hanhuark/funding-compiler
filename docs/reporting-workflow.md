# Reporting Workflow

The current report writer produces Markdown so outputs remain easy to inspect, edit, and convert.

## One-Page Briefs

A one-page brief can be generated by filtering to one opportunity and keeping the top faculty matches. The useful sections are sponsor, deadline, award size, topic summary, top matches, and rationale.

## Detailed Reports

A detailed department report can include all active opportunities, ranked faculty matches, topic counts, sponsor counts, and notes for proposal planning. Markdown reports can be converted to PDF, DOCX, or slides using tools such as Pandoc, Quarto, or a future report exporter.

## Review Loop

The match score is a starting point, not a final decision. Research administrators or faculty leads should review matched keywords, eligibility constraints, deadline fit, and team balance before acting on recommendations.
